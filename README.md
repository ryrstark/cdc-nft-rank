# Crypto.com - NFT Rank displayer

Simple and unofficial extension used to display the rank of some nft within collections on the nft platform of crypto.com.

Collection supported yet:
* PsychoKitties
* Loaded Lions
* PsychoMolly
* Alphabot
* Alphabot Armory Weapon
* Ballies
* BeatHeadz
* MadHare
